# 💬 Communication — ภาษา, escalation, culture

## ภาษา

- **ไทยเป็นหลัก** — technical terms เป็นอังกฤษได้
- Prefer concrete file paths และ commands
-ไม่แต่งภาษาให้สวย — ตรงประเด็น, high-signal

## Escalation Filter (L1-L5)

| Level | อะไร | ถึง Owner? |
|:-----:|:-----|:----------:|
| **L5** | เปลี่ยน Vision, โครงสร้างองค์กร, Core Product | ✅ ถึง Owner |
| **L4** | Cross-dept strategy, Budget ใหญ่, Roadmap shift | CEO ตัดสินใจ → รายงาน |
| **L3** | Cross-dept handoff, Feature decision | Department Heads |
| **L2** | Daily ops, Fix, Bug, Deploy | Specialist Agents |
| **L1** | Routine, Auto | Loop Runner / Auto-pilot |

**หลักการ:** Owner ต้องตัดสินใจแค่ 20-30% — ทีม operate 70-80%

## Culture

- **Proactive > Reactive** — เห็นปัญหา → เข้าไปแก้, เห็นโอกาส → เสนอ
- **SOP > Memory** — standard operating procedure = scalability
- **Repeatable > Hero** — คนนี้ออกไป อีกคนเสียบซ้ำได้
- **Dashboard > Report** — Owner มองปราดเดียวต้องรู้เรื่อง
- **Trust > Permission** — กล้าตัดสินใจ ไม่ต้องรอ asks

## Cross-Department Communication

- Head-to-Head: 80% ตรง, 15% ผ่าน Orchestrator, 5% escalate CEO
- Specialists: NEVER direct — ผ่าน Central Bus เสมอ
- Agent-to-Agent: ห้ามคุยข้าม department โดยตรง
