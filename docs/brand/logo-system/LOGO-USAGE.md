# SoloCorp OS — Logo Usage Guidelines

## Primary Logo
`logo-primary.svg` — Full lockup with Hexagon Nexus mark + "SoloCorp OS" wordmark + tagline

**When to use:**
- Website header, docs site, landing page
- GitHub README, social media banners
- Presentation title slides
- Print materials

---

## Icon Only
`logo-icon.svg` — Hexagon Nexus mark standalone (72×64)

**When to use:**
- App icon, browser tab icon
- PWA manifest icon
- Social media profile picture
- Navigation bar (when space is tight)
- Pipeline UI status indicator

---

## Favicon
`logo-favicon.svg` — Simplified S-monogram in hexagon (16×16)

**When to use:**
- Browser tab favicon
- Bookmark icon
- Small badge contexts

---

## Clear Space
- Minimum clear space = height of "S" in SoloCorp
- No other elements within this zone
- For icon-only: clear space = 25% of icon width

## Minimum Size
- Full logo: 120px width
- Icon only: 24px width
- Favicon: 16px

## Color Usage
- On dark bg (#0A0A0B): use logo-primary.svg (white + indigo)
- On light bg: request inverted version from Design
- On brand accent (#6366F1): use monochrome white version
- Never recolor the logo

## What NOT to Do ❌
- ❌ Don't stretch or distort
- ❌ Don't change colors
- ❌ Don't add shadows or effects
- ❌ Don't rearrange elements
- ❌ Don't use on low-contrast backgrounds
- ❌ Don't outline or stroke the mark
