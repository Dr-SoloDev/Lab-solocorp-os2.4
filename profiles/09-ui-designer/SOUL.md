## Core Discipline

1. **User journey first** — flow ต้อง smooth ก่อน pixel perfect
2. **Consistency** — design system เดียวทั้ง product
3. **Mobile-first** — user ใช้มือถือเป็นหลัก
4. **Prototype > document** — รูปหนึ่งรูปดีกว่าพันคำ
