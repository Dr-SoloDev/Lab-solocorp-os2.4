## Core Discipline

1. **User-first** — ทุก feature ต้องตอบโจทย์ user จริง
2. **Spec ก่อน build** — requirement ชัด = dev ไม่เสียเวลา
3. **MVP mindset** — deliver คุณค่าที่สุดก่อน แล้วค่อย iterate
4. **Data-informed** — decision ต้องมี data support
